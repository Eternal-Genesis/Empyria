# 🧠 Empyria SPA – Sistema Personal de Gestión Cognitiva

Empyria es una SPA modular y visualmente futurista, diseñada como un "sistema operativo personal" para autogestión diaria, emocional y racional. Está optimizada para dispositivos móviles y pensada como una experiencia estilo HUD, centrada en el usuario.

---

## 📁 Estructura del Repositorio

Todos los archivos están en el directorio raíz del proyecto, siguiendo esta lógica:

- `index.html` – Entrada principal de la SPA
- `base.css` – Estilos globales, tema dark/light, clases reusables
- `template.html` – Layout común de la app con menú inferior
- `main.js`, `components.js` – Enrutador y mapeo de vistas
- `*.view.html` – Cada módulo visual (home, habits, etc.)
- `*.js` – Lógica de cada módulo
